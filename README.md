# 🎬 IMDb Movies & TV Shows Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** built using IMDb Movies & TV Shows dataset. The goal of this project is to analyze trends, ratings, genres, languages, and global distribution of movies and TV shows.

The dashboard is divided into two major sections:

* **Exploration Dashboard**
* **Analysis Dashboard**

It helps users quickly gain insights into the entertainment industry using dynamic filters and visualizations.

---

## 📊 Dashboard Features

### 🔍 1. Exploration View

This section focuses on high-level insights:

* 🎥 **Top 5 Best Movies & TV Shows**

  * Based on IMDb ratings
* 🎭 **Top 5 Genres**

  * Drama, Comedy, Romance, Thriller, Action
* 🌍 **Top 5 Languages**

  * English, French, Japanese, Spanish, German
* 🌎 **Releases by Country**

  * Interactive map showing global distribution

---

### 📈 2. Analysis View

This section focuses on deeper insights:

* 📅 **Number of Releases Each Year**

  * Trend analysis of content growth over time
* ⭐ **Rating vs Votes Correlation**

  * Scatter plot showing relationship between popularity and ratings
* 📊 **Rating Trends Over Years**

  * Understanding how ratings evolved historically
* 🧩 **Category Distribution**

  * Movies vs TV Episodes comparison

---

## 🎛️ Filters Available

The dashboard includes interactive slicers for:

* Genre
* Country
* Year
* Rating
* Language

These filters allow users to perform dynamic analysis based on their preferences.

---

## 🛠️ Tools & Technologies Used

* **Power BI** – Data visualization & dashboard creation
* **Excel / CSV Dataset** – Data source
* **Data Cleaning & Transformation** – Power Query
* **DAX (Data Analysis Expressions)** – Calculations & measures
